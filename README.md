<h1 align='center'>TuitionSoftware 🎲</h1>

## 📋 Descripción 📋

ToutionSoftware es un panel de usuario que te muestra la matrícula a los cursos de tu preferencia, como también los ya matriculados.

## 👨‍💻 Tecnologías 👨‍💻

- Backend
	- Express.js (Node.js)
	- Mysql
	- Prisma (ORM)
	- Passport.js (Authentication)
	- JWT (Authentication)
	- ...

- Frontend
	- React.js
	- react-router-dom
	- luxon
	- ...

## 📦 Estructura del proyecto 📦
	Iniciar Sesión
		-> Formulario de Inicio de Sesión
	Registro
		-> Formulario de Registro
	Panel de Estudiante
		-> Cursos
			- Cursos matriculados
		-> Matrícula
			-> Cursos Disponibles
			-> Horarios Disponibles
			-> Lista de cursos escogidos
			-> Modal: Confirmación de matrícula

##  🌌 Pasos a seguir 🌌
```bash
	git clone https://github.com/grupo-desarollo-web-fullstack/toution-software.git
	cd toution-software
	# if you use backend
	cd backend
	# if use frontend
	cd frontend
	npm install
	npm run dev
```
